# SysML Diagrams (PlantUML)

This folder contains SysML-style diagrams for the **Intelligent Fraud Prevention Dashboard** (synthetic-only).

## How to render

- VS Code: install the "PlantUML" extension (jebbs.plantuml)
- Ensure Java is installed (or use the extension's server mode)
- Open any `.puml` file and use **PlantUML: Preview Current Diagram**

## Diagram index

- Activity: `activity.puml`
- Block Definition (BDD): `bdd.puml`
- Internal Block (IBD): `ibd.puml`
- Package: `package.puml`
- Parametric: `parametric.puml`
- Requirement: `requirements.puml`
